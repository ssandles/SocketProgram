Chat Application with Python Socket Programming

This repository contains Python code for a simple chat application using socket programming. It consists of both client and server scripts that enable multiple clients to connect to a central server and exchange messages.
Getting Started

To run the chat application after cloning repository to machine, follow these steps:

In terminal navigate to the repository directory:
" cd <repository_name> "

Run the server script:

"python server.py" or "python3 server.py"

Run the client script in separate terminal windows or environments:

"python client.py" or "python3 client.py"

Follow the prompts to choose a nickname and start chatting!!! 


Features

   * Nickname Selection: Clients can choose their nicknames upon connection.
   * Real-time Communication: Messages are exchanged in real-time between clients and the server.
   * Error Handling: The application handles errors gracefully, ensuring stable communication.

Technologies Used

   * Python
   * Socket Programming
   * Threading
